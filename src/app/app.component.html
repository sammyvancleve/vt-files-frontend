This website is built to help Vermonters identify where information they might want to know would be held in State of Vermont agenices. Under the Vermont Public Records Law and the Freedom of Information Act, any Vermonter can legally request public files from any local, state, or federal agency. In order to use the website, just ask the below search bar any question or query about Vermont the same way you would Google. The website will provide the relevant Vermont state agency for you to request files from.
